Overview

21M.385 uses Kivy - an application / graphics framework for python. We also use a few other packages: numpy, pyaudio (which uses portaudio) and fluidsynth. Note that this class uses python3.

Follow the steps below to install everything you need for Mac, Windows, or Linux.

Mac Installation Instructions

  • Install the LeapMotion System using the Mac installer found here.
  • Download the latest Python using miniconda. Select the python 3.7 64-bit (bash installer) for Mac OSX.
  • Install miniconda by opening up a terminal window, going to the location of the installer, and typing 
    $ bash Miniconda3-latest-MacOSX-x86_64.sh
  • Close this terminal window since it will not have the correct path to the newly installed python.
  • Get osx.zip, the osx install package and unzip it.
  • Open a terminal window and cd to the osx installation directory (for example, if the osx package is in Downloads type cd Downloads/osx).
  • Run the installer (it uses homebrew, conda install, and pip install): 
    $ bash install_osx.sh
  • This installation will downgrade your python to 3.6.5. This is important because later versions of python have an incompatibility with Leap Motion.
  • Please note if any errors occurred. If so, contact me or post on Piazza to get help.
  • Test the installation by running: 
    $ python3 check_install.py

Windows Installation Instructions

  • Install LeapMotion using the Windows installer found here.
  • Install Python using miniconda. Select the 64-bit python 3.7 for Windows.
    • When you install miniconda, I recommend installing for "Just Me". Otherwise, you may have permission issues in the installation process.
    • Additionally, check the box that adds Anaconda to your PATH (even though this is "not recommended" by the installer).
  • Get the ASIO drivers. The default windows audio drivers are often bad - exhibiting a lot of latency.
  • Get the win install package and unzip it.
  • Open a new Command Terminal.
  • From the cmd line, cd to the win directory you just unzipped and run:
 > install_win.bat

This might take a while. It will also downgrade your python installation to 3.6.5, which is necessary for the rest of the installer to run properly. Note if any errors occurred. If so, contact me or post on Piazza to get help.

  • Test the installation by running:
 > python check_install.py

Linux Installation Instructions

  • Linux is not very well supported for this class. It is preferable to use Mac or Windows. However, if that is not an option, you should install the following:
  • Install python 3.6.5. As with Windows or Mac, you may want to use miniconda.
  • Install Leapmotion
  • Install Fluidsynth
  • Install Portaudio
  • Install Kivy
  • Install the following python modules:
    • numpy
    • matplotlib
    • python-rtmidi
    • pyaudio
  • Test the installation by running (same code as the Mac or Win checker above):
 > python check_install.py
